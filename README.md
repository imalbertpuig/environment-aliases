First of all, install Neovim: https://neovim.io/

# BASH

```
vim ~/.bash_profile
```

Refresh aliases without rebooting:

```
source ~/.bash_profile
```

# ZSH

```
vim ~/.zshrc
```

Refresh aliases without rebooting:

```
source ~/.zshrc
```
